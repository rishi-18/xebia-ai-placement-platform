## `1. Student User Stories` 

## `US-01` 

```
As a student, I want to create and manage my profile so that recruiters can view
my qualifications and skills.
```

## `US-02` 

```
As a student, I want to upload my resume so that I can apply for placement
opportunities.
```

## `US-03` 

```
As a student, I want AI-based job recommendations so that I can find jobs that
match my skills and interests.
```

## `US-04` 

```
As a student, I want to track the status of my applications so that I can stay
informed about my placement progress.
```

## `US-05` 

```
As a student, I want AI-generated interview preparation questions so that I can
improve my interview performance.
```

## `US-06` 

```
As a student, I want notifications about new placement drives so that I do not
miss any opportunities.
```

## `2. Recruiter User Stories` 

## `US-07` 

```
As a recruiter, I want to create company profiles so that students can learn
about my organization.
```

## `US-08` 

```
As a recruiter, I want to post job openings so that eligible students can apply.
```

## `US-09` 

```
As a recruiter, I want AI-powered candidate recommendations so that I can
quickly identify suitable applicants.
```

## `US-10` 

```
As a recruiter, I want to filter candidates based on skills, CGPA, branch, and
experience so that I can shortlist candidates efficiently.
```

## `US-11` 

```
As a recruiter, I want to schedule interviews through the platform so that the
recruitment process becomes easier to manage.
```
